Welcome to h3-OpenSYS/WaitIO(WaitIO) documentation!
===================================

**h3-Open-SYS/WaitIO** (WaitIO for short) is a communication library
 to couple multiple MPI programs among heterogeneous systems.

WaitIO provides an inter-program communication environment among MPI
programs and supports different MPI libraries with various
interconnects and processor types.
    
Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
